# Gaming Set

Source: Player's Handbook (2024)

*Tool (Gaming Set)*

- **Cost:** Varies (1-5 gp)
- **Weight:** Varies (0.5-2 lb.)

Includes items like Three-Dragon Ante cards, Chess, or Dice. Wielding a Gaming Set allows you to make ability checks with proficiency to play games, challenge opponents, or perform strategic maneuvers.
